# Derek Personal Website Preview

This is a static preview website for Dr Derek Guotao Ma and the Stochastic Risk Intelligence Lab.

## How to preview locally

Open `index.html` directly in a browser, or run:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Files

- `index.html` — homepage content and structure
- `styles.css` — visual design and responsive layout
- `script.js` — small scroll animation and dynamic year
